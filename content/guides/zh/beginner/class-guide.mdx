## 决策框架

选择职业不是选"最强"的——而是将职业与你当前的装备状况、刷图目标和对风险的容忍度匹配。按以下框架判断：

1. **武器路径**——你能持续升级主手武器吗？不能保证武器进度的职业就是死路。
2. **副手和防御**——职业的天然生存能力（HP、护甲、盾牌）是否匹配你正在推进的内容难度？
3. **属性对齐**——你可用的材料和效果是否支持这个职业的优先属性？
4. **阶段适配**——这个职业适合你当前所处的游戏阶段吗（早期、中期、后期）？

## 职业分析

### 起手三职业怎么选

如果只看开局，选择可以压缩成一个表：

| 职业 | 适合谁 | 核心风险 | 什么时候换 |
|---|---|---|---|
| 骑士 | 完全新手、想稳定推进的人 | 清图慢 | 你已经熟悉关卡并且想提高材料/金币产量 |
| 游侠 | 已经能稳定通关、想提速的人 | 防御薄、依赖弓和箭等级 | 你开始频繁死亡或武器跟不上 |
| 法师 | 懂材料和属性堆叠的人 | 脆、吃 Cube 和冷却属性 | 你已经有足够材料支撑法系路线 |

这里没有“永远正确”的职业。职业是工具，当前目标变了，最优工具也会变。

### 骑士——基准职业

骑士是游戏中的参考职业，有充分理由。高 HP、重甲和盾牌让你在别的职业会死的情况下活下来。用骑士来：
- 学习关卡布局、怪物伤害模式和 BOSS 机制
- 判断你的装备等级是否足够应对当前内容
- 在投入脆皮职业之前测试新关卡
- 比较刷图路线时建立金币/经验基准

**为什么不永远用骑士？** 清图速度。一旦你熟悉关卡并有装备支撑，伤害型职业会清得更快。但骑士在高难度内容和多英雄配置中仍然是安全网。

### 游侠——效率之选

游侠用生存换清图速度。弓和箭提供稳定的远程伤害，同时比近战更安全。它是免费职业中最好的：
- 需要数量（材料、金币、经验）的中期刷图
- 远程持续输出比爆发更重要的关卡
- 想要更快清图但还没准备好面对法师或 DLC 职业复杂性的玩家

**游侠的弱点：** 较低的防御属性意味着装备等级不够的游侠死得很快。武器进度（弓+箭升级）至关重要——弓不够强的游侠只是慢慢等死。

### 法师——投资之选

法师奖励那些理解属性堆叠和材料效果的玩家。它造成范围伤害、施加 debuff（减速），并从不均等受益于冷却缩减和施法速度。但它很脆，需要数值支撑才能发挥作用。

**什么时候转法师：** 当你有足够材料并能用 Cube 堆叠法师属性（法系伤害、冷却、施法速度），并且足够了解关卡可以安全走位。

**什么时候不要转：** 捡到一根高稀有法杖之后。没有支撑属性的稀有武器只会产生令人失望的结果。

### 祭司——倍增器

祭司是唯一的专职辅助职业且**免费 DLC**。它的治疗和复活能力防止团灭并延长刷图时长。在多英雄配置中，祭司让你的伤害输出英雄效果倍增。

**Solo 祭司**可行但清得很慢。真正的威力在配合受益于治疗时长和攻击 buff 的伤害职业时才能体现。把祭司当作你的第二或第三个英雄，而不是第一个。

#### 祭司怎么加点和组队

公开资料对祭司的共识很清楚：它不是主输出，而是队伍稳定器。优先级按这个顺序：

1. **Power Blessing**——优先提升，核心价值是放大队伍输出。
2. **Sanctuary**——第二优先，用治疗和续航减少失败局。
3. **Max HP**——祭司自己活着，治疗和增益才存在。
4. **Cooldown Reduction**——缩短关键技能间隔，让队伍少出现真空期。

免费三英雄队伍的合理终点是 **骑士 + 游侠 + 祭司**：骑士抗伤害，游侠打稳定远程输出，祭司提供治疗、复活和攻击增益。如果购买 Hunter DLC，可以把游侠替换成猎人做更高输出，但这不是新手默认路线。

### 猎人（DLC）——伤害天花板

猎人是游戏中伤害最高的职业。十字弓带爆炸和闪电 AOE、元素状态堆叠、卓越的 BOSS 伤害。但是：
- 付费 DLC——免费玩家不可用
- 需要装备投入才能达到上限
- 相比骑士较脆

**建议：** 在你真正投入游戏并有装备准备时再买猎人。不要把它当作新手的首选职业。

### 屠戮者（DLC）——风险之选

屠戮者通过牺牲 HP 来打爆发伤害。成功时令人兴奋，失败时惩罚严重。大多数指南把它排在同一前排角色的骑士之下，因为：
- 自我伤害意味着不完美操作就会更多死亡
- 免费骑士在同一角色定位中更可靠
- HP 牺牲需要具体装备来抵消

**建议：** 把屠戮者当作后期实验，不是核心刷图职业。在投入资源之前需要强证据级别的 Build。

## 避免单件装备的陷阱

一把高稀有武器不等于职业推荐。如果你捡到一件 Legendary 法杖，不代表你应该转法师。那根法杖需要：
- 副手武器在可用等级
- 能在相关关卡站住脚的护甲
- 堆叠法师属性的材料和效果
- 证明转职合理的清图时间

换职业之前，先问："我能用新配置稳定清理目标关卡吗？" 如果答案不明确，留着物品，等待。

## 三英雄槽位路线

多英雄不是装饰，它会改变职业选择。第二槽让你从“单职业硬推”变成“双职业互补”，第三槽让祭司的价值真正释放出来。

1. 英雄升到 3 级后打开 Runes。
2. 优先拿早期成长节点，再推进到解锁第二英雄槽的命令类节点。
3. 第二槽先放游侠或骑士，用来补你主职业缺的稳定性或输出。
4. 第三槽是长期金币目标，解锁后再把祭司放进队伍。
5. 不要为了凑三英雄牺牲装备等级。三个低等级英雄不如两个能稳定清图的英雄。

这个路线的判断标准不是“槽位越多越好”，而是每个新槽位是否让你的死亡率下降、清图时间下降，或让更高难度变成稳定可刷。

## 正确阅读 Build 页面

本站的 Build 页面标注了证据等级：
- **数据挖掘**——从提取的游戏数据构建，可信度高
- **编辑整理**——基于游戏知识策划的推荐
- **社区**——常见玩家路线，未经单独验证
- **未验证**——进行中，仅作灵感参考

没有哪个 Build 是"最强"的。把 Build 当检查清单用：核心属性目标、装备方向、材料效果优先级和关卡适配。始终用自己的清图时间验证。
